  DevConnect 🚀

DevConnect is a collaborative social ecosystem designed for developers to network, showcase technical projects, and leverage AI-powered tools to enhance code comprehension and content creation.
🛠 Tech Stack

    Backend: Python, Django

    Database: PostgreSQL, Firebase (Auth & OTP)

    Design & Prototyping: Figma (UI/UX)

    AI Integration: LLaMA (LLM) for intelligent code interpretation

    Deployment: Render

✨ Key Features

    Recommendation Engine: A custom-built algorithm mapping user specializations to suggest relevant professional connections.

    Smart Personalized Feed: Content prioritization based on historical user engagement.

    AI-Driven Assistant: * Real-time technical code interpretation.

        AI-powered content optimization for posts.

        Automated tag extraction and categorization.

    Secure Authentication: Robust account management featuring OTP-based password recovery.

    High-Fidelity Design: Full UI/UX workflow implemented via Figma prototypes.
